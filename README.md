## Overview 
Real-time chat application with WebSocket communication. 
 
## Features 
- Real-time messaging 
- Group chats 
- Online status 
- Message history 
- File sharing 
 
## Tech Stack 
- **Backend:** Node.js, Socket.io 
- **Frontend:** React.js 
- **Database:** MongoDB
